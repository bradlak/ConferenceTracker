# ConferenceTracker

Application for tracking conference in Xamarin.Forms
</br>

It uses:
* Xamarin.Forms
* Unity IoC container
* Connectivity plugin
* ImageCircle plugin
* MVVM pattern
* MVVM Light Toolkit
